# 🚀 Quick Start Guide

Get running in **5 minutes**.

## 4 Steps

### 1️⃣ Copy Template
```bash
cp .env.example .env.local
```

### 2️⃣ Add API Key
Edit `.env.local`:
```env
VITE_YOUTUBE_API_KEY=your_api_key_here
```

Need a key? See [SETUP.md](SETUP.md)

### 3️⃣ Install
```bash
npm install
```

### 4️⃣ Run
```bash
npm run dev
```

Open http://localhost:8000 → Done! 🎉
